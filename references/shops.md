# Shops

## Tools

| Tool | What it does |
|------|-------------|
| **Shop Rankings** | Top shops by revenue, sales, growth, product count, affiliate/self-operated/mall revenue split |
| **Shop Detail** | Full metrics for specific shops: revenue trends, channel structure, creator count, product mix |

## Ranking Sort Options

revenue, sales_volumn, revenue_growth_rate, unit_price, on_sell_product_count, affiliate_revenue, self_operated_account_revenue, shopping_mall_revenue

## Ranking Filters

| Filter | Values | Example |
|--------|--------|---------|
| category_ids | Category ID list | Shops in a specific category |
| revenue_range | Revenue range string | Filter by revenue tier |
| unit_price_range | Price range string | Filter by average order value |
| shop_type | "BRAND" / "RETAILER" | Brand stores vs retailers |
| keyword | Shop name search | Search by shop name |

## Date Ranges

- Rankings: lastDay, last7Day, last30Day
- Detail: lastDay, last7Day, last30Day, last90Day, last180Day, last365Day
