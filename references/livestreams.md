# Livestreams

## Tools

| Tool | What it does |
|------|-------------|
| **Livestream Rankings** | Top livestreams by revenue, views, duration, start/end time |
| **Livestream Detail** | Full metrics for specific livestreams: product list, viewer metrics, revenue breakdown |

## Ranking Sort Options

revenue, unit_price, views, livestream_duration, livestream_start_time, livestream_end_time

## Ranking Filters

| Filter | Values | Example |
|--------|--------|---------|
| category_ids | Category ID list | Livestreams in a specific category |
| shop_id | Shop ID | Livestreams from a specific shop |
| creator_id | Creator ID | Livestreams by a specific creator |
| product_id | Product ID | Livestreams featuring a specific product |
| followers_range | Follower range string | Livestreams by creators of a certain size |

## Date Ranges

- Rankings: lastDay, last7Day, last30Day
- Detail: lastDay, last7Day, last30Day, last90Day, last180Day, last365Day
