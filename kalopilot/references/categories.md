# Categories

## Tools

| Tool | What it does |
|------|-------------|
| **Category Rankings** | Top categories by revenue, growth rate, average revenue per shop |
| **Category Detail** | Full metrics for specific categories: revenue trends, shop count, product count. Only supports last7Day/last30Day |
| **Category Search** | Find category IDs by keyword. Returns id, name, and level (1/2/3) |

## Ranking Sort Options

revenue, revenue_growth_rate, average_revenue

## Ranking Filters

| Filter | Values | Example |
|--------|--------|---------|
| category_ids | Category ID list | Filter to specific categories |
| category_level | "1" / "2" / "3" | Top-level, mid-level, or leaf categories |
| revenue_range | Revenue range string | Filter by category revenue tier |

## Date Ranges

- Rankings: lastDay, last7Day, last30Day, last180Day
- Detail: last7Day, last30Day only
