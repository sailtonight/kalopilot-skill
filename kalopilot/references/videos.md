# Videos

## Tools

| Tool | What it does |
|------|-------------|
| **Video Rankings** | Top videos by revenue, views, engagement (likes, shares, comments), ad ROAS |
| **Video Detail** | Full metrics for specific videos: revenue trends, view count, engagement breakdown |
| **Video Script Extraction** | Frame-by-frame script with timeline, camera angles, visuals, and voiceover text. **100 credits/video**, cached 7 days |

## Ranking Sort Options

revenue, views, revenue_growth_rate, ads_roas, digg_count, share_count, comment_count, ad_revenue_ratio, ad_view_ratio

## Ranking Filters

| Filter | Values | Example |
|--------|--------|---------|
| category_ids | Category ID list | Videos in a specific category |
| shop_id | Shop ID | Videos from a specific shop |
| creator_id | Creator ID | Videos by a specific creator |
| product_id | Product ID | Videos promoting a specific product |
| revenue_range | Revenue range string | High-revenue videos only |
| followers_range | Follower range string | Videos from creators of a certain size |
| ads_roas | ROAS value or range | High-performing ad videos |

## Date Ranges

- Rankings: lastDay, last7Day, last30Day
- Detail: lastDay, last7Day, last30Day, last90Day, last180Day, last365Day
