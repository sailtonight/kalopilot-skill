# Creators / Influencers

## Tools

| Tool | What it does |
|------|-------------|
| **Creator Rankings** | Top creators by revenue, GMV growth, followers, content views, video/live revenue split |
| **Creator Detail** | Full metrics for specific creators: revenue trends, follower analysis, content performance |
| **Creator Search** | Search TikTok for creators by name or handle. Returns top matches with recent videos |

## Ranking Sort Options

revenue, revenue_growth_rate, content_views, content_views_growth_rate, creator_followers, sales_volumn, video_revenue, live_revenue

## Ranking Filters

| Filter | Values | Example |
|--------|--------|---------|
| category_ids | Category ID list | Creators in a specific niche |
| shop_id | Shop ID | Creators working with a specific shop |
| product_id | Product ID | Creators promoting a specific product |
| revenue_range | Revenue range string | Filter by GMV tier |
| creator_type | "BELONGED_TO_SELLER" / "INDEPENDENT" | Shop-owned accounts vs independent creators |
| followers_range | Follower range string | Micro/mid/macro influencers |
| engagement_rate | "LOW" / "MEDIUM" / "HIGH" | Filter by engagement level |
| keyword | Creator name or ID | Search by name |

## Date Ranges

- Rankings: lastDay, last7Day, last30Day
- Detail: lastDay, last7Day, last30Day, last90Day, last180Day, last365Day
